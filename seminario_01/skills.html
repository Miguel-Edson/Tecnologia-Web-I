Habilidades

A seguir, apresento brevemente as principais ferramentas de desenvolvimento e design com as quais já tive a oportunidade de trabalhar e aplicar em projetos anteriores — mesmo que, em alguns casos, ainda em nível básico ou em constante evolução.


  1. Next.js

    Tenho experiência no desenvolvimento de aplicações web com Next.js, aproveitando seus recursos de renderização server-side (SSR), rotas dinâmicas e otimização de performance.
    Utilizo a estrutura para construir interfaces modernas, com carregamento rápido e excelente experiência para o usuário. Também aplico boas práticas de SEO e integração com APIs.


  2. Git

    Utilizo Git no controle de versão dos meus projetos, garantindo organização, rastreabilidade e colaboração eficiente em equipe. Com o GitHub, gerencio repositórios, crio branches,
    abro pull requests e participo de fluxos colaborativos, além de manter meu portfólio de projetos acessível e atualizado.


  3. CSS3

    Utilizo CSS3 para criar interfaces responsivas, modernas e agradáveis visualmente, com domínio em flexbox, grid, animações, transições e media queries. Também aplico princípios de
    design visual, como hierarquia, espaçamento e contraste, garantindo uma ótima experiência em diferentes dispositivos.


  4. HTML5

    Tenho domínio da estruturação semântica de páginas web utilizando HTML5, garantindo acessibilidade, organização e compatibilidade com boas práticas de SEO. Uso tags apropriadas para
    criar uma base sólida e clara para qualquer tipo de aplicação web.


  5. JavaScript

    Trabalho com JavaScript moderno para desenvolver interações dinâmicas, manipular o DOM, consumir APIs e aplicar lógica de negócio em aplicações web. Tenho familiaridade com funções
    assíncronas (async/await), eventos, e estruturação de código limpo e reutilizável.


  6. Figma

    Utilizo o Figma para criação de protótipos interativos, designs responsivos e organização de sistemas visuais. Trabalho com componentes reutilizáveis, grids e tipografia, garantindo
    que o design esteja alinhado com a experiência do usuário e seja facilmente replicável no código.
